# Проект: Место

Интерактивная страница с возможностью:
 * редактировать профиль
 * добовлять, удалять фотографии
 * ставить лайки

### Технологии
* HTML5
* CSS3
* Nested BEM
* Flexbox
* Grid Layout
* UX Lean Validation
* Javascript
